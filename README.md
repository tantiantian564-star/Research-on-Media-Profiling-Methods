## 二、专业巩固

考虑到升学的需要，在老师的建议下，本周重点研读了发表于《中国图象图形学报》的综述论文《类脑赋能视觉增强：原理、方法与前沿进展》（余磊等，2025）。该文系统梳理了以事件相机为代表的类脑视觉感知技术在视频增强任务中的理论原理、方法演进与未来趋势。

**核心收获总结**：

- **事件相机的工作原理**：事件相机仿效生物视网膜，仅感知场景光强的动态变化，输出异步事件脉冲，具有μs级时间分辨率、120dB高动态范围和极低功耗，弥补了传统帧相机在高速、高动态场景下的不足。
- **三类增强范式**：
  - 时域补偿：利用事件的高时间分辨率，解决运动去模糊、视频插帧、卷帘快门校正等任务；
  - 空域补偿：结合事件与图像的空间关联，实现超分辨率重建；
  - 联合增强：同时利用高时间分辨率与高动态范围，应对多种降质（模糊+低动态+卷帘畸变）的复杂场景。
- **技术瓶颈**：低光照下事件流噪声与时间戳扰动、高分辨率下的海量事件处理延迟、算法实时性不足等仍是当前难点。
- **对本人毕设的启发**：文中“多模态融合”和“跨模态信息补偿”的思想对我的媒体画像构建有一定的迁移价值——在媒体画像中，可借鉴类似思路融合文本、行为、社交等多源异构数据，提升画像鲁棒性。
